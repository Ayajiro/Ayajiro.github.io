Vuepress component 改寫

Tue Dec 15 2020
  • Vue.js

自己搜尋Vue.js方法改寫成component的時候遇到一些問題，因爲對Vue.js概念還不是很清楚，做個筆記。

<div id="counter">
  Counter: {{ counter }}
</div>

Vue.js

const Counter = {
  data() {
    return {
      counter: 0
    }
  }
}

Vue.createApp(Counter).mount('#counter')

Vuepress

export default {
  data() {
    return {
      counter: 0
    }
  }
}

result:

Counter: 0
